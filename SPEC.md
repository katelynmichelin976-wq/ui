# PWA 交互按钮 - 规范文档

## 1. 项目概述
- **项目名称**: Victory Buttons PWA
- **项目类型**: 移动端优先的交互式网页应用
- **核心功能**: 两个具有丰富反馈效果的按钮 - 振动反馈和音效+烟花庆祝
- **目标用户**: 移动端用户

## 2. UI/UX 规范

### 布局结构
- 全屏单页应用，垂直居中布局
- 顶部标题区域（安全距离内）
- 中央两个按钮垂直排列
- 烟花效果层叠在按钮之上，铺满整个屏幕

### 视觉设计
- **背景**: 深色渐变背景 `#0a0a1a` → `#1a1a3a`，营造庆祝氛围
- **主色调**:
  - 完成按钮: `#2dd4bf` (青绿色)
  - 通关按钮: `#f59e0b` (金黄色)
- **辅助色**:
  - 按钮悬停: 亮度提升 10%
  - 文字: `#ffffff`
- **字体**: "Noto Sans SC", sans-serif
- **按钮尺寸**: 宽度 200px，高度 56px，圆角 28px
- **间距**: 按钮间距 24px

### 组件规范

#### 按钮状态
- 默认: 实色背景，柔和阴影
- 悬停: 亮度提升，轻微上移 transform: translateY(-2px)
- 激活: 缩放 transform: scale(0.95)

#### 烟花粒子
- 颜色: 金色 `#ffd700`、红色 `#ff4757`、蓝色 `#70a1ff`、绿色 `#7bed9f`、紫色 `#a55eea`
- 粒子数量: 每次点击 80-120 个粒子
- 动画时长: 2秒
- 粒子大小: 3-8px 随机

## 3. 功能规范

### 完成按钮
- **振动反馈**: 使用 navigator.vibrate(50) 产生 50ms 短振动
- **视觉反馈**: 按钮产生波纹扩散效果
- **iOS 兼容**: iOS 不支持振动API，静默失败

### 通关按钮
- **音效**: 使用 Web Audio API 生成收银机音效（快速上升音阶）
- **烟花效果**:
  - 从按钮中心向四周迸射
  - 粒子受重力影响下落
  - 粒子逐渐淡出
- **屏幕闪烁**: 轻微的背景闪烁增强庆祝感

### PWA 支持
- manifest.json 配置
- 可添加到主屏幕
- 全屏显示
- 离线可用

## 4. 验收标准

- [ ] 两个按钮在手机上有良好的触摸响应
- [ ] 完成按钮点击时产生可感知的振动（Android）
- [ ] 通关按钮点击时播放收银机音效
- [ ] 通关时屏幕出现烟花庆祝动画
- [ ] 可以添加到主屏幕作为 PWA
- [ ] 适配各种手机屏幕尺寸
